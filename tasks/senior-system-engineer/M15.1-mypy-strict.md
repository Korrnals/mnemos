## Категории ошибок (45 шт.)

### Категория A — `attr-defined` (4 errors) — **HIGH priority**

`src/mnemos/manager.py:33-36`:
```python
if TYPE_CHECKING:
    from mnemos.pipeline.cluster import ClusterResult   # ← mypy не видит
    from mnemos.pipeline.publish import PublishResult
    from mnemos.pipeline.quality_gate import QualityResult
    from mnemos.pipeline.synthesize import SynthesisResult
```

mypy говорит: `Module "mnemos.pipeline.cluster" does not explicitly export attribute "ClusterResult"`.

**Fix**: добавить в `mnemos/pipeline/__init__.py`:
```python
from mnemos.pipeline.cluster import ClusterResult, cluster_memories
from mnemos.pipeline.publish import PublishResult, publish_memory
from mnemos.pipeline.quality_gate import QualityResult, evaluate_quality
from mnemos.pipeline.synthesize import SynthesisResult, synthesize_cluster
__all__ = [...]  # явно
```

### Категория B — `Memory | None → Memory` (2 errors) — **HIGH priority**

`src/mnemos/manager.py:223,230`:
```python
memory = await self.sqlite.get(...)  # type: Memory | None
# ... позже
self.something(memory)  # mypy: variable has type "Memory"
```

**Fix**: добавить явный None-check или assert с сообщением. Не подавлять через `cast()` без обоснования.

### Категория C — `object` типы в vault.py (6 errors) — **MEDIUM priority**

`src/mnemos/storage/vault.py:103-108`:
```python
meta = post.metadata  # type: object (frontmatter.Post.metadata is Any)
created_at=_parse_dt(meta.get("created")),  # mypy: object has no attribute get
```

**Fix** (вариант 1, минимальный):
```python
meta: dict[str, Any] = post.metadata  # type: ignore[assignment]  # upstream lib untyped
```

**Fix** (вариант 2, правильный):
```python
from typing import cast
meta = cast("dict[str, Any]", post.metadata)
```

Предпочитаем вариант 2.

### Категория D — `no-any-return` (15+ errors) — **LOW priority, скучно**

`src/mnemos/storage/sqlite_store.py:544,563,575`:
```python
def counts(self) -> dict[str, int]:
    return conn.execute(...).fetchone()  # type: Any
```

**Fix**: добавить type guard:
```python
def counts(self) -> dict[str, int]:
    row = conn.execute(...).fetchone()
    if row is None:
        return {}
    return dict(row)  # или явный dict comprehension с int(value)
```

Аналогично в:
- `src/mnemos/storage/vector_store.py:47,142,150`
- `src/mnemos/embeddings/__init__.py:92,99,168,176,203,206`

### Категория E — `policy/scheduler.py:90-99` (6 errors) — **HIGH priority**

```python
# problem: object instead of dict
value = self._job_store[job_id]  # mypy: object
if value["status"] == "completed":  # mypy: object is not indexable
    self.dlq_discard(job_id, value["id"])  # mypy: object is not str
```

Это **потенциальный runtime NoneType crash** если `_job_store` хранит что-то неожиданное.

**Fix**:
```python
job: dict[str, Any] | None = self._job_store.get(job_id)
if job is None:
    continue  # или log warning
status = str(job.get("status", ""))
if status == "completed":
    job_id_str = str(job.get("id", ""))
    self.dlq_discard(job_id, job_id_str)
```
