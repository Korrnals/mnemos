## Findings (детально)

### B608 — SQL injection (3 места) — **HIGH**

#### Finding 1: `src/mnemos/storage/sqlite_store.py:419`
```python
def update_fields(self, memory_id: str, **kwargs: Any) -> bool:
    allowed = {"status", "quality_score", "confidence", ...}
    updates = {k: v for k, v in kwargs.items() if k in allowed}
    setters = ", ".join(f"{k}=?" for k in updates)  # ← B608
    values = [*updates.values(), memory_id]
    cur = conn.execute(f"UPDATE memories SET {setters} WHERE id=?", values)
```

**Анализ**: bandit не знает, что `setters` собирается из whitelisted `allowed`. Сейчас безопасно, но **anti-pattern** — любая будущая правка allowlist может сломать безопасность.

**Fix** (вариант 1, type-safe):
```python
_FIELD_UPDATERS: dict[str, str] = {
    "status": "status=?",
    "quality_score": "quality_score=?",
    "confidence": "confidence=?",
    "source_coverage": "source_coverage=?",
    "cluster_id": "cluster_id=?",
    "derived_from": "derived_from=?",
    "embedding_id": "embedding_id=?",
    "clean_content": "clean_content=?",
    "filter_profile": "filter_profile=?",
    "filter_stats": "filter_stats=?",
    "filter_version": "filter_version=?",
    "title": "title=?",
    "content": "content=?",
    "tags": "tags=?",
    "category": "category=?",
    "file_path": "file_path=?",
}

def update_fields(self, memory_id: str, **kwargs: Any) -> bool:
    updates = {k: v for k, v in kwargs.items() if k in _FIELD_UPDATERS}
    if not updates:
        return False
    for field in ("derived_from", "tags", "filter_stats"):
        if field in updates and not isinstance(updates[field], str):
            updates[field] = json.dumps(updates[field], ensure_ascii=False)
    setters = ", ".join(_FIELD_UPDATERS[k] for k in updates)
    values = [*updates.values(), memory_id]
    conn = self._get_conn()
    cur = conn.execute(
        f"UPDATE memories SET {setters}, updated_at=? WHERE id=?",  # nosec B608
        [*values, datetime.now(UTC).isoformat()],
    )
    return cur.rowcount > 0
```

`# nosec B608` здесь допустим **только** с обоснованием "setters built from whitelisted _FIELD_UPDATERS dict".

#### Finding 2: `src/mnemos/storage/sqlite_store.py:523`
```python
def fts_search(self, ...):
    sql = f"""
        SELECT ... FROM memories_fts f
        JOIN memories m ON m.id = f.id
        WHERE 1=1 {category_filter} {tag_filter} {project_filter} ...
    """
```

**Анализ**: 5-6 f-string интерполяций в одном SQL. Каждая — потенциальный injection.

**Fix**: вынести каждое условие в именованную переменную, использовать parameter substitution:
```python
def fts_search(
    self,
    query: str,
    *,
    limit: int = 10,
    project: str | None = None,
    agent: str | None = None,
    status: MemoryStatus | None = None,
) -> list[Memory]:
    conn = self._get_conn()
    fts_query = self._build_fts_query(query)  # escape quotes!
    where_parts: list[str] = ["memories_fts MATCH ?"]
    params: list[Any] = [fts_query]
    if project:
        where_parts.append("m.project = ?")
        params.append(project)
    if agent:
        where_parts.append("m.agent = ?")
        params.append(agent)
    if status:
        where_parts.append("m.status = ?")
        params.append(status.value)
    where_clause = " AND ".join(where_parts)
    sql = f"""
        SELECT m.* FROM memories m
        JOIN memories_fts f ON m.rowid = f.rowid
        WHERE {where_clause}
        ORDER BY rank
        LIMIT ?
    """  # nosec B608 — where_parts are static + parametrised
    params.append(limit)
    return [self._row_to_memory(r) for r in conn.execute(sql, params).fetchall()]

@staticmethod
def _build_fts_query(user_query: str) -> str:
    """Escape FTS5 special chars: \" ' * ( )."""
    # Strip FTS5 operators from user input, replace with whitespace
    safe = re.sub(r'[^\w\s\u0400-\u04FF-]', ' ', user_query)
    # Wrap in double quotes to disable FTS5 syntax
    return f'"{safe}"'
```

### B615 — HF Hub download без pin (3 места) — **HIGH (supply chain)**

`src/mnemos/embeddings/__init__.py:133,135,137`:
```python
from huggingface_hub import hf_hub_download
model_path = hf_hub_download(repo_id="sentence-transformers/all-MiniLM-L6-v2", filename="onnx/model.onnx")
# ↑ нет revision=, что значит может скачать compromised model при push
```

**Fix**:
```python
# В конфиге (config.py) добавить:
class EmbeddingSettings(BaseSettings):
    hf_revision: str = "c9745ed1d7e3b0194c2e1c2b5d7e3e0b3c1c1c1c"  # pin конкретного commit SHA
    # или pin branch: hf_revision: str = "main"  # минимально, но SHA лучше
```

```python
# В embeddings/__init__.py:
model_path = hf_hub_download(
    repo_id="sentence-transformers/all-MiniLM-L6-v2",
    filename="onnx/model.onnx",
    revision=settings.embedding.hf_revision,  # pinned
)
```

**Нужно выбрать**: tag (например "v1.0") или commit SHA. SHA надёжнее (immutable), но менее читаемо. Рекомендация: **SHA** для production.

**Дополнительно**: добавить SHA256 verify после скачивания. Если HF Hub когда-нибудь скомпрометируют — verification всё равно поймает.

### B104 — bind to 0.0.0.0 (1 место) — **false positive для контейнера**

`src/mnemos/manager.py:379` (если есть в HTTP API startup):
```python
uvicorn.run(app, host="0.0.0.0", port=8000)
```

**Fix**: оставить `0.0.0.0`, но добавить обоснование:
```python
# Container/podman runs in a private network namespace, but we want to
# accept connections from the host's bridge interface. Binding to 0.0.0.0
# is intentional and required for the container port-mapping to work.
uvicorn.run(app, host="0.0.0.0", port=8000)  # nosec B104 — container networking
```

Это **подтверждённый false positive** для контейнерного деплоя. Bypass через inline `# nosec` с обоснованием — допустимо по `lint-and-validate.instructions.md` §"When suppression is acceptable".

**Альтернатива**: запускать uvicorn из CLI с флагом `--host 0.0.0.0`, и в коде дефолт `127.0.0.1`. Тогда bandit вообще не увидит.
